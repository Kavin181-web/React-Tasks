# React Tasks

This project is a simple React application that demonstrates basic student operations using JavaScript and React.

In this project, I have implemented different tasks such as displaying students, updating data, adding new students, deleting students, and filtering data based on conditions.

Concepts used:
- React.js
- JavaScript
- Array methods (map, filter)
- Conditional logic

Features:
- Display all students
- Update a student's status
- Add a new student
- Delete a student
- Filter students based on age and course
- Check if status exists
